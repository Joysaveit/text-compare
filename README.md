# 文本对比工具 (Text Comparison Tool)

一个基于Web的字符级文本对比工具，支持实时差异检测、智能导航和同步滚动功能。

## 🚀 功能特性

### 核心功能
- **字符级差异检测**：使用Myers算法精确识别文本差异
- **双向对比**：同时显示原始文本和修改文本
- **智能语义清理**：自动优化差异显示效果

### 导航功能
- **快速定位**：一键跳转到任意差异位置
- **顺序浏览**：上一处/下一处差异导航
- **差异列表**：下拉菜单快速选择
- **实时计数**：显示差异总数和当前位置

### 用户体验
- **同步滚动**：左右结果框基于字符位置的精确同步
- **宽屏适配**：1400px最大宽度，支持大屏幕显示
- **大字体显示**：12px字体，250px输入框，600px结果区域
- **响应式设计**：适配不同屏幕尺寸

## 🛠️ 技术栈

- **前端**：纯HTML5 + CSS3 + JavaScript
- **差异算法**：diff-match-patch库
- **架构**：无依赖，可直接在浏览器中运行

## 🎯 使用方法

### 直接运行
1. 双击 `index.html` 在浏览器中打开
2. 或右键 → "打开方式" → 选择浏览器

### 本地开发
```bash
# 克隆项目
git clone [你的仓库地址]

# 本地运行
# 直接打开 index.html 即可使用
```

## 🎮 操作指南

1. **输入文本**：在左右文本框中输入要对比的文本
2. **开始对比**：点击"开始对比"按钮
3. **查看差异**：
   - 红色：删除的内容
   - 绿色：新增的内容
   - 灰色：相同的内容
4. **导航差异**：
   - 使用"上一处"/"下一处"按钮
   - 或直接输入差异编号跳转
   - 从下拉列表选择差异
5. **历史记录**：
   - 点击"保存记录"保存当前对比会话
   - 点击"历史记录"查看和管理历史对比记录
   - 支持加载历史记录继续编辑

## 🔄 同步滚动

左右结果框支持基于字符位置的精确同步滚动：
- 滚动任意一侧，另一侧自动同步到对应字符位置
- 支持内容长度差异较大的情况
- 防抖处理，避免频繁触发
