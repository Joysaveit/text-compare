# Joy Comparison Chrome扩展 - 快速安装指南

## 🚀 一键安装步骤

### 1. 打开Chrome扩展管理页面
在Chrome浏览器地址栏输入：`chrome://extensions/`

### 2. 开启开发者模式
点击页面右上角的"开发者模式"开关

### 3. 加载扩展
点击"加载已解压的扩展程序"按钮，选择当前文件夹

### 4. 开始使用
点击Chrome工具栏中的Joy Comparison图标即可使用

---

## 📁 必需文件清单

确保以下文件都在同一文件夹中：
- ✅ `manifest.json` - 扩展配置
- ✅ `background.js` - 后台脚本  
- ✅ `index.html` - 主页面
- ✅ `style.css` - 样式文件
- ✅ `script.js` - 核心逻辑
- ✅ `dmp.min.js` - 差异算法库
- ✅ `icons/` - 图标文件夹

## 🎯 功能预览

- **独立窗口**：点击图标打开1400x900像素窗口
- **字符级对比**：精确检测文本差异
- **智能导航**：快速跳转到任意差异位置
- **历史记录**：保存和管理对比历史
- **同步滚动**：左右结果框精确同步

## 🔧 故障排除

**问题**：扩展无法加载
**解决**：检查所有必需文件是否完整，重新加载扩展

**问题**：图标不显示  
**解决**：使用 `generate_icons.html` 生成PNG图标

**问题**：功能异常
**解决**：确保Chrome版本88+，检查控制台错误信息
